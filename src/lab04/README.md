## Цель работы

Изучить:

- интерфейсы
- абстрактные классы (ABC)
- полиморфизм через единый интерфейс

---

## Реализованные интерфейсы

### Printable

Интерфейс для вывода объектов.

Метод:

```python
display()
```

---

### Calculable

Интерфейс для вычислений.

Метод:

```python
calculate_income()
```

---

## Реализация интерфейсов

Интерфейсы реализованы в классах:

- `Apartment`
- `ResidentialApartment`
- `CommercialApartment`

---

## Полиморфизм

Методы интерфейсов работают по-разному в разных классах.

Пример:

```python
obj.calculate_income()
```

Каждый класс вычисляет доход по своей логике.

---

## Использование интерфейса как типа

```python
def print_all(items: list[Printable]):
```

Функция может работать с любыми объектами, реализующими интерфейс `Printable`.

---

## Проверка через isinstance()

Используется:

```python
isinstance(obj, Printable)
```

Для проверки реализации интерфейса.
