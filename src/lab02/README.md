# ЛР-2. Коллекция объектов

## Предметная область
Недвижимость.

## Используемый класс
Используется класс `Apartment` из ЛР-1.

---

## Коллекция

```python
class ApartmentCollection:
    ...
```

`ApartmentCollection` — контейнер для хранения объектов `Apartment`.

---

## Назначение

Класс позволяет работать с группой квартир:

- добавлять и удалять объекты  
- получать список всех квартир  
- искать, сортировать и фильтровать  
- итерироваться по коллекции  

---

## Хранение данных

Внутри используется список:

```python
self._items = []
```

---

## Основные методы

- `add(item)` — добавление  
- `remove(item)` — удаление  
- `remove_at(index)` — удаление по индексу  
- `get_all()` — получить список объектов  

---

## Поиск

- `find_by_title()` — по названию  
- `find_by_address()` — по адресу  

Методы возвращают новую коллекцию.

---

## Сортировка

- `sort_by_price()`  
- `sort_by_area()`  
- `sort(key)`  

---

## Фильтрация

- `get_available()` — доступные квартиры  
- `get_expensive(min_price)` — дорогие  
- `get_large(min_area)` — большие  

---

## Магические методы

- `__len__` → `len(collection)`  
- `__iter__` → `for item in collection`  
- `__getitem__` → `collection[0]`  
- `__str__` → красивый вывод  

---

## Ограничения

- только объекты `Apartment`  
- нельзя добавить дубликат  
- корректная работа удаления  

---

## Демонстрация

В `demo.py` показано:

- добавление объектов  
- вывод коллекции  
- поиск  
- `len()` и `for`  
- индексация  
- сортировка  
- фильтрация  
- удаление  

---

## Итог

Реализован контейнер `ApartmentCollection`, который:

- хранит объекты  
- управляет ими  
- поддерживает поиск, сортировку и фильтрацию  
- позволяет итерироваться и работать как со списком
