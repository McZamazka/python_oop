# Реализованный класс

## Apartment

```python
class Apartment:
    property_type = "Жилая недвижимость"
```

Главный класс, моделирующий объект недвижимости (квартиру).  
Содержит всю основную логику работы с квартирой.

## Атрибут класса

- `property_type` — тип недвижимости (общий для всех объектов)

Используется для демонстрации разницы между атрибутом класса и экземпляра.

---

## Закрытые поля

- `_title` — название квартиры  
- `_area` — площадь  
- `_price` — цена аренды  
- `_address` — адрес  
- `_rent_months` — срок аренды  
- `_status` — статус  

Это инкапсуляция — данные защищены и не изменяются напрямую.

---

## Свойства (@property)

- `title` — получение названия  
- `area` — получение площади  
- `price` — чтение и изменение (через setter)  
- `address` — получение адреса  
- `rent_months` — чтение и изменение (через setter)  
- `status` — получение статуса  

Позволяют безопасно работать с данными.

---

## Валидация

- `validate_title()` — проверка названия  
- `validate_area()` — проверка площади  
- `validate_price()` — проверка цены  
- `validate_address()` — проверка адреса  
- `validate_rent_months()` — проверка срока аренды  
- `validate_status()` — проверка статуса  

Проверяется:
- тип данных  
- пустые строки  
- положительность значений  
- допустимость статуса  

Вся логика вынесена в отдельный файл `validate.py` и не дублируется.

---

## Магические методы

- `__init__` — создание объекта  
- `__str__` — красивый вывод через print  
- `__repr__` — технический вывод  
- `__eq__` — сравнение объектов  

Делают объект удобным для работы и вывода.

---

## Бизнес-методы

- `monthly_price_per_m2()` — цена за 1 м²  
- `total_rent_cost()` — общая стоимость аренды  
- `rent()` — сдача квартиры  
- `archive()` — отправка в архив  
- `reopen()` — возвращение из архива  

Реализуют основную логику работы с квартирой.

---

## Логические состояния объекта

- `available` — доступна  
- `rented` — сдана  
- `archived` — в архиве  

Поведение объекта зависит от состояния:

- нельзя сдать квартиру, если она уже сдана  
- нельзя сдать квартиру, если она в архиве  
- нельзя архивировать квартиру, если она сдана  
- открыть можно только архивную квартиру
